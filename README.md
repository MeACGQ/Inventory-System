Unity Inventory & Interaction System

Hi, I'm Ahmet Can (MeACGQ).
Thanks for checking out this project.

I built this project to understand how an inventory system works in Unity and how it connects with player interaction and UI. I tried to keep the system simple but also flexible enough to expand later.

# Features
- Item pickup system
- Hotbar inventory (1–5 slots)
- Stackable items
- Item dropping system
- Item icons in UI
- Highlight system for interactable objects
- What I focused on

# While working on this project, I mainly focused on:

- Inventory logic and slot management
- Implementing a stack system
- Using raycasts for interaction
- Connecting gameplay systems with UI
- Keeping the system modular and reusable
